# Custom Vehicle Painter for Cherax 5.46.1
A small and simple tool to play around with vehicle colors :)

Total beginner here! Im just starting out with Lua and GTA scripting. This project is basically a personal learning aid to help me get the hang of things

### üé® Features
* Base Material: Select between Metallic, Matte, Chrome, Pure, Brushed and 32 Chameleon colors
* Color Mixer: Adjustments for Brightness, Saturation, and Hue Shift
* Sync: Synchronization of Primary, Secondary, Neon, Xenon, and Tyre Smoke colors (No Wheel colors)
* Pearl: Calculates the Pearlescent ID based on your current base color
* Randomizers: Generate paintjobs or chameleon effects fast

### üìÅ Sharing & Presets
All settings are saved in a JSON file, making it easy to share your creations with the community
* File Location: `C:\Users\{USERNAME}\Documents\Cherax\Vehicles\Paintjobs\`
* Format: `.json` files

How to share: 
1. Go to the folder path above
2. Copy the `.json` file and send it to others
3. To use a shared file, simply place it in the same folder and use the Load function in-game
4. You can also modify these files and load them in-game

üíÅ Installation:
1. Put `Paintjobs.lua` in your `C:\Users\{USERNAME}\Documents\Cherax\Lua\` folder
2. Load the script in-game
